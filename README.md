# Crypto Volume Analysis Toolkit

A simple but practical toolkit that helps you track high-volume crypto tokens in the last 24 hours and generate cross-market (spot & futures) analysis reports in a PDF file. 

The toolkit works on all devices.

## The toolkit includes:

- Spot Volume Tracker v2.0
- Advanced Futures + Spot Analysis v1.0
- Automatic HTML report generator
- Automatic PDF export
- Cleanup system that removes source files after analysis
